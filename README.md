# Product CRUD System

## Tech Stack

- React (Vite)
- Node.js
- Express.js
- MongoDB Atlas

## Features

- User Registration
- Login Authentication
- Add Product
- Edit Product
- Like Product
- View Liked Products
- User Profile
- Edit Profile

## Installation

### Backend

```bash
cd srs-backend
npm install
npm start
```

### Frontend

```bash
cd srs-frontend
npm install
npm run dev
```
